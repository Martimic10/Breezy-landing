# Breezy Landing Page

A modern, premium landing page for Breezy — a mobile app for splitting expenses with friends.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Tech Stack

- **Next.js 15** — React framework
- **Tailwind CSS 4** — Styling
- **Framer Motion** — Animations
- **Lucide React** — Icons

## Sections

- Hero with iPhone mockup and floating UI elements
- Scroll-based feature showcase with sticky phone transitions
- How It Works steps
- Pricing (Free + Pro)
- FAQ accordion
- Final CTA with green gradient
- Footer
